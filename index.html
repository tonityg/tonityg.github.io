Encore aujourd'hui, je continue d'apprendre beaucoup de chose sur le développement informatique dans la programmation web, de jeux vidéos ou encore de logiciels. J'ai eu l'occasion de développer plusieurs projets que ce soit personnels ou avec des amis qui m'a permis d'acquérir des compétences dans le travail d'équipe notamment.

Sur mon Portfolio, vous pourrez trouver certains de mes projets que j'ai pu aboutir et quelques uns qui n'ont jamais été terminé mais qui m'ont permis d'approfondir dans ce domaine qu'est l'informatique. Merci d'avoir prêté attention à mon Portfolio et je vous souhaites une bonne visite.

Stuff Industry

Stuff Industry est le deuxième jeu crée à l'occasion d'une Game Jam (une sorte de compétition où l'objectif est de créer un jeu vidéo en un minimum de temps en respectant un thème et des règles précises). Le premier étant "Patch The Bugs".

Ce jeu a donc été réalisé en 3 jours à l'aide du moteur de jeu Unity durant la Mini Jam 113 dont le thème était "Boss Final". En plus de faire parti de mes premiers jeux créés pour des Game Jam, il s'agit aussi d'un de mes premiers jeux "terminé" en général. J'en suis assez fier même si graphiquement je pouvais difficilement faire mieux à mon niveau mais c'était une très bonne expérience.

Patch The Bugs

Il s'agit du premier jeu que j'ai réussi à "terminer". Il a été crée durant la Lost Relic Game Jam 2022 avec comme thème "Connexion". Une game jam est une compétition où le but est de créer un jeu vidéo en un temps indiqué limité sur un thème donné. Ici, nous avions 9 jours pour faire ce jeu (mais je l'ai rejoint en retard donc je n'avais que 3 ou 4 jours).

Ce jeu a été fait durant ma première Game Jam (le deuxième étant "Stuff Industry") est a donc été ma première expérience dans la création de jeu sous la pression de trouver l'idée, et de créer le jeu dans un temps très limité.

J'ai été assez fier après avoir publié le jeu puisqu'il s'agit de mon premier jeu où j'ai pu réunir toutes les connaissances que j'ai obtenu même si il est particulièrement pauvre graphiquement et que le gameplay reste assez limité.

Mastermind

A l'origine, ce projet consistait à concevoir un Mastermind qui fonctionne sur une console Python et donc devait fonctionner qu'avec des caractères du clavier. J'ai ensuite refais ce projet (d'abord fait en Python) sur le moteur de jeu Godot en me basant sur le code du projet en version Python (fait au départ). Le jeu est fait pour fonctionner sur Android mais peut être tester sur PC en ouvrant le code source disponible sur mon compte Github avec Godot.
